### First CLI program written in rust

Simple todo list used with CLI.

## List of commands and subcommands
	1. `todo` - returns list of todo items
	2. `todo new  --message -m` - create new todo item
	3. `todo del --number -n` - delete todo item with given number (arg is required)
	4. `todo done --number -n` mark item as done
	5. `todo undone --number -n` mark item as undone
